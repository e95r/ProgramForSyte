# ProgramForSyte

Локальное desktop-приложение для ведения соревнований по плаванию: импорт старт-листов из Excel, распределение по заплывам, фиксация неявок, ввод результатов и формирование протоколов.

## Возможности
- Импорт стартового протокола из `.xlsx` в локальную SQLite-базу (`./meet/meet.db`).
- Просмотр дистанций и участников с назначенными заплывами/дорожками.
- Отметка неявок (**Не явился**) и возврат спортсменов в заплывы (**Вернуть в заплывы**).
- Пересев участников:
  - мягкий (в пределах текущих заплывов),
  - полный (с возможной перенумерацией заплывов).
- Ввод результатов по дистанции с маской времени `00:00:00` и судейскими отметками (`DNS`, `DQ`, `EXH` и др.).
- Формирование протокола дистанции и итогового протокола соревнований:
  - группировка,
  - сортировка,
  - печать,
  - сохранение в `PDF`, `HTML`, `TXT`.
- Ручной и автоматический бэкап базы данных.

## Требования
- Python 3.11+ (код совместим минимум с 3.10)
- PySide6
- openpyxl
- pytest

## Краткое руководство использования

### 1) Импорт старт-листа
1. Нажмите **Импорт Excel**.
2. Выберите файл стартового протокола (`.xlsx`).
3. После успешного импорта в левой панели появится список дистанций.

> Приложение запоминает последний выбранный файл и автоматически создаёт бэкап при следующем запуске (если база уже существует).

### 2) Просмотр дистанций и поиск участников
1. В блоке **Дистанции** выберите нужную дистанцию.
2. В правой части используйте поле **Поиск по ФИО** для фильтрации списка.
3. Для массовых операций выделяйте несколько строк (MultiSelection).

### 3) Работа с неявками
- **Не явился** — ставит выбранным спортсменам статус DNS.
- **Вернуть в заплывы** — снимает DNS и сразу выполняет пересев.
- Чекбокс **Полный пересев**:
  - выключен: мягкий пересев,
  - включен: полный пересев (приложение запросит подтверждение).

### 4) Пересев вручную
1. Выберите дистанцию.
2. При необходимости включите **Полный пересев**.
3. Нажмите **Пересобрать**.

### 5) Ввод результатов
1. Выберите дистанцию.
2. Нажмите **Ввод результатов**.
3. Заполните колонку **Результат** в формате `мм:сс:сс` (маска `00:00:00`).
4. При необходимости заполните **Отметка** (`DNS`, `DQ`, `EXH` и т.п.).
5. Нажмите **Сохранить результаты**.

### 6) Протоколы и печать

#### Протокол дистанции
1. Выберите дистанцию.
2. Нажмите **Протокол дистанции**.
3. Настройте группировку/сортировку и обновите отображение.
4. Используйте:
   - **Печать** — отправка на принтер,
   - **Сохранить** — экспорт в `PDF`, `HTML` или `TXT`.

#### Итоговый протокол соревнований
1. Нажмите **Подвести итоги**.
2. Выберите нужный режим группировки/сортировки.
3. Распечатайте или сохраните файл.

### 7) Резервное копирование
- Нажмите **Бэкап БД** для создания ручной копии базы.
- Если база уже есть, при старте приложения создаётся автоматический бэкап.
